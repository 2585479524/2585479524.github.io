大数开方--牛顿迭代法

本文总共1271个字 | 您是第位看到它们的小伙伴

如果一个数的位数为偶数个，那么这个数开方之后就有n/2位；

如果一个数的位数为奇数个，那么这个数开方之后就有n/2+1位；

我们以num=1000为例，位数为4，偶数，那么开方之后就有2位。我们从高位开始枚举。

首先枚举十位：

10*10=100 < 1000

20*20=400 < 1000

30*30=900 < 1000

40*40=1600 > 1000

则十位数字为3

枚举各位：

31*31=961 < 1000

32*32 =1024 > 1000

则个位数字为1

所以开方1000的结果为31。
